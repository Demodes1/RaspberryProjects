¿Cómo Funciona?

El Usuario que no tenga credencial apretara un botón donde se activara el micrófono y la cámara para poder comprobar su identidad.

El Usuario podra decir su matrícula y si es reconocida, se le dara su acceso. También si la cámara reconoce su cara, se le dara acceso.

Sensores Utilizados

RaspBerry

Botón: Dara inicio al Sistema

Cámara: Reconocera caras que esten en la base de datos

Led (Simulación de Puerta): Si Enciende, simula el acceso

Pantalla LCD: Mostrara Datos

Micrófono: El Usuario podrá decir su matrícula
